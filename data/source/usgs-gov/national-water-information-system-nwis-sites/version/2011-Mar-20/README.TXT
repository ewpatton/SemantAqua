There are several scripts in the directory.
Here are how to use them.

0. install BeautifulSoup, which is available at http://www.crummy.com/software/BeautifulSoup/
* http://www.crummy.com/software/BeautifulSoup/download/3.x/BeautifulSoup-3.2.0.tar.gz
* tar xzf
* cd BeautifulSoup-3.2.0/
* bash-3.2$ python setup.py install

1. ./to-source-usgs-sites.sh StateAbbr, e.g. RI. For now, we only support CA, MA, RI, NY
